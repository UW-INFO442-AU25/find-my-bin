# Find My Bin

Find My Bin helps users quickly identify the correct waste bins for everyday items by combining fast item recognition with simple, interactive learning that develops long-term sorting habits and reduces uncertainty at sorting stations. By making disposal choices clearer and more intuitive, it reduces recycling contamination, supports responsible consumption, and encourages more sustainable practices. 


## Features

<ul>
  <li>
    <b>Quick Sort:</b> Users can navigate through categories, item groups, materials, and conditions to receive a recommended bin based on the item they are sorting. 
    
  </li>
  <li>
    <b>AI Image Recognition:</b> Users can upload an image of an item to receive a bin recommendation.
  </li>
  <li>
    <b>Sorting Quiz:</b> Users can practice sorting waste items by answering timed questions, earning points for correct answers, and reviewing their question history to track progress over time. 
  </li>
  <li>
    <b>Registration & Log-in:</b> Tracks lifetime quiz points and answer history. 
  </li>
</ul>
